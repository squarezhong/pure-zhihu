# Pure Zhihu

从问答平台到社交媒体（不带任何推荐算法的那种）。

## 简介

知乎网页端简化脚本：

- 默认从首页进入关注流
- 在全站移除“关注”栏之外的顶栏噪音
- 移除“知乎直答”按钮
- 侧边栏移除“大家都在搜”和“盐言作者平台”
- 由于知乎赚钱挺不容易的，所以不对显式的广告做任何过滤。

### 模式

- 扩散模式：默认
- 严格模式：在扩散模式的基础上移除所有“xxx赞同了xx”

## 安装

需要浏览器装有 Tampermonkey 或 Violentmonkey 插件。

[安装 Pure Zhihu (Github)](https://raw.githubusercontent.com/squarezhong/pure-zhihu/main/pure-zhihu.user.js)
